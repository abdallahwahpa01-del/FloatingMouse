# FloatingMouse Android Application

**Description:**
This project provides a floating mouse-like experience for Android devices, featuring an interactive cursor and joystick controls for seamless interaction across all applications. It implements:
- WindowManager for overlays
- AccessibilityService for system interaction
- Joystick UI for smooth cursor movement

---

**Features:**
1. **Floating Cursor:** Works over all applications.
2. **Left and Right Click Simulations:** Mimics mouse click and context menu operations.
3. **Joystick Control:** Move the cursor in all directions seamlessly.
4. **Customizable Settings:**
   - Enable/Disable floating mouse.
   - Adjust cursor size and speed.
